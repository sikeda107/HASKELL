# Haskell
## 実行方法
### GHC
`user$ ghc Hello.hs`
`user$ ./Hello hoge huga`
### GHCi
```
user$ ghci
Prelude> :load Hello
*Main> :main hoge huga
```
### スクリプト
`user$ runghc Hello.hs hoge huga`
### QuickCheckパッケージ確認
`user$ ghc-pkg list QuickCheck`
